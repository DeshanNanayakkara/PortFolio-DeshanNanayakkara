GreenShadow Crop Monitoring System

Project Overview

Green Shadow crop monitoring system is designed to streamline farm management processes and enhance operational efficiency through a user-friendly interface and robust backend functionalities. It enables effective monitoring and management of various aspects of farming operations, providing a seamless solution for modern agricultural needs.

Backend Development

  • Built with: Java Spring Boot, ensuring a scalable and maintainable architecture.
  • Security: Implements JWT (JSON Web Token) Security for secure authentication and authorization.
  • Data Persistence: Managed using MySQL, ensuring efficient and reliable storage.

Frontend Development

  • Designed with: HTML, CSS, and JavaScript for a responsive experience.
  • Framework: Utilizes Bootstrap for a clean and modern interface.

Key Features

  • Staff Management: Add, update, and monitor staff details.
  • Vehicle Management: Track and manage farm vehicles efficiently.
  • Field Monitoring: Record and view field data, including codes, locations, and crops.
  • Image Uploads: Attach and manage field-related images.
  • Secure Access: Role-based access control powered by JWT.
